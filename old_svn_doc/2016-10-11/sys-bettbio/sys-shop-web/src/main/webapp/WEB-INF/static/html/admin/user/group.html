组ID 名称 权限
1 SUPERADMIN 只有一个用户可以是超级管理员，管理员用户是被默认创建作为管理员的。这个用户可以在系统中做任何事，并是唯一可以管理商家的人。
2 ADMIN 管理员用户拥有管理商家的最高权限。一个管理员用户可以做商家经理所能做的所有事，并且可以创建其他权限受限的账户。
3 ADMIN_CATALOGUE 本组用户可以创建商品、类别和特别推荐。
4 ADMIN_STORE 本组用户可以管理被授权的商家。他能够改变品牌和其他任何在商家主菜单上的项目，并且拥有发货和支付的配置权限。
5 ADMIN_ORDER 本组用户可以管理订单、查找订单、修改客户信息、申请退款。
